How to Use
Encontre qualquer product em segundos.
1
Digite o código ou nome do product
No campo de busca da tela inicial, digite o código (ex: QQ.RH.L.46) ou o nome do product. Os resultados aparecem instantaneamente.
2
Use autocomplete to speed things up
Enquanto digita, sugestões aparecem automaticamente. Toque em uma sugestão para ir direto ao product.
3
Scan the packaging (optional)
Toque em "Scan product name" para abrir a câmera. Aponte para o nome ou código impresso na etiqueta ou embalagem. Não é necessário código de barras.
4
Go to the indicated location
The location is highlighted in the result. Example: H5 03 indicates column H, level 5, position 03.
5
Found an error?
Let Thiago know immediately so the information can be corrected and the system stays reliable.
About Parisi360
Sistema interno de localização de products.

The Parisi360 is an internal system developed exclusively for the Parisi Bathware warehouse team.

Seu único objetivo é ajudar os funcionários a encontrar a location física dos products de forma rápida, precisa e eficiente — eliminando o tempo perdido procurando itens no armazém.

This system is not an e-commerce, is not connected to Harmoniq, and does not control stock, orders or any other commercial operation.
